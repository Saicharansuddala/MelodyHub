🎵 MelodyHub

MelodyHub is a full-stack music platform that delivers a smooth and interactive audio experience. It allows users to explore music, manage playlists, and enjoy a responsive, modern interface backed by a scalable backend.

---

🚀 Features

- 🎧 Music streaming with smooth playback
- 🔍 Search songs, artists, and albums
- ❤️ Like and save favorite tracks
- 📂 Create and manage playlists
- 🔐 Secure authentication system
- ⚡ Fast and responsive UI
- 🌐 Cross-device compatibility

---

🛠️ Tech Stack

Frontend

- React.js
- Tailwind CSS
- Vite

Backend

- Node.js
- Express.js

Database

- MongoDB

⚙️ Setup Instructions

Install dependencies

Frontend

cd frontend
npm install

Backend

cd backend
npm install

---

▶️ Run the Project

Start Backend Server

cd backend
npm start

Start Frontend

cd frontend
npm run dev

---

🔐 Environment Variables

Create a ".env" file inside the backend folder:

MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
